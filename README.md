# 🖼️ 希米相册管理系统 (v5.01)

这是一个轻量级、高效且功能丰富的 PHP 相册管理系统。支持 3D 相册生成、多相册管理、防盗链及图片访问统计等功能。系统完全基于文件存储，不依赖任何数据库，极易部署。

## 🚀 项目特性

* **运行环境**：原生支持 **PHP 7.0 - 8.5** 核心环境。
* **零数据库**：完全不使用任何数据库，支持虚拟主机/虚拟机搭建，代码即传即用。
* **3D 相册生成**：优化了 3D 相册生成步骤，支持配置多个带有独立后台的 3D 相册。
* **图片高级处理**：
  * 支持常见格式图片上传，系统自动转换为高性能的 **WebP** 格式，并生成自定义缩略图。
  * 批量导入 3D 相册时支持多种格式自动转换（*注意：3D 相册手动单独上传时仅支持 `.jpg` 格式*）。
* **安全与隐私保护**：
  * 支持设置**私密相册**，支持多相册、多分类独立后台管理。
  * 全局支持**防盗链**与**防盗链白名单**访问，可有效防止第三方工具恶意爬取或下载。
* **数据流统计**：支持图片外链分享，内置完善的图片访问日志统计与分析功能。

## 🛠️ 后台管理与初始配置

### 1. 超级管理员后台
* **登录地址**：`/admin/login.php` （亦可从首页右上角菜单直接进入）。
* **修改初始密码**：打开根目录 `/index_src/ini/setting.php`，将默认密码 `admin` 修改为你的专属强密码。

### 2. 默认模板配置说明
所有默认模板的核心文件均位于 `/index_src/default/` 目录下：

* **相册单独管理后台 (`login.php`)**：
  * 默认账号：`ximi`
  * 默认密码：`【随机值】`
  * *注：为避免粗心忘记修改初始密码，系统已将其设为安全随机值。*
* **相册访问日志 (`log.php`)**：
  * 默认初始密码：`admin` （请按需修改或关闭）。
  * **彻底关闭日志功能**：打开 `img.php` 文件，删除日志记录核心函数后续的所有代码即可。
* **相册白名单配置 (`functions.php`)**：
  * 用于手动添加和修改你的防盗链白名单域名（亦可在超级管理后台直接可视化修改）。
* **默认配置文件 (`ini.php`)**：
  * 单个相册的核心配置文件，支持手动通过代码修改高级行为。

## ⚙️ 配置文件说明 (`ini.php`)

在自定义或迁移相册时，你可以通过修改单个相册目录下的 `ini.php` 来调整路径和属性。标准的配置数组结构如下：

```php
$config = array (
  'min'         => '../../update/user/cheng/class/min_image/',  // 小缩略图存放路径
  'max'         => '../../update/user/cheng/class/max_image/',  // 大缩略图存放路径
  'tu_1'        => '../../update/user/cheng/class/img/',        // 原图存放路径
  'login_admin' => 'login_default',                             // 该相册的登录隔离标识
  'list_read'   => '1',                                         // 相册隐私权限：0 = 私密，1 = 公开
  'title'       => '城',                                        // 相册名称/标题
  'cover'       => '',                                          // 相册封面图 URL 地址（留空则随机获取）
  'list_home'   => '1',                                         // 首页可见性：0 = 隐藏，1 = 展现
  'txt'         => '',                                          // 相册文字介绍（在相册底部展示）
);
```
